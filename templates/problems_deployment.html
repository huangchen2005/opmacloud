Web服务器的选择

通过前面的介绍，我们基本上选定了租用云主机才承载我们的服务，由于云主机类似一个真实的机器，所以上面的服务都是需要我们自己来配置的。 既然是web服务，首先我们需要Web服务器，Web服务器的选择我们一般选用Nginx，因为Nginx性能好，消耗内存少，支持更多并发连接,支持负载均衡和反向代理， 我们基本已经用Nginx替代了Apache。

应用服务器的选择

应用服务器的选择与应用使用的语言有关。一般热门的Web语言有Java、Php、Python等。

Java

一般情况Java容器我们都用开源的Tomcat，需要注意的是要做好Tomcat的监控，包括Java内存信息的监控。

PHP

如果是PHP程序的话，一般我们用PHP-fpm来进行处理。PHP版本尽量选用最新的版本，别忘了开启opcache,他可以在不改代码的情况下加速PHP站点。

Python

如果您的网站是用Python写成的话，建议您supervisord来管理python进程。

数据库的选择

一般情况下我们用的都是Mysql。我们可以选择将Mysql直接搭建在云主机上，也可以选择云服务商的DB服务，这取决于您对性能和价格的要求。
